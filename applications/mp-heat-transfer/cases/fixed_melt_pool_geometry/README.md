# Laser-induced heating of a 2D fixed melt pool surface geometry

Benchmark example from [^1].

## Simulations

### diffuse

```bash
mpirun -np 64 /path/to/applications/mp-heat-transfer/mp-heat-transfer input_diffuse_CE_fine.json
mpirun -np 64 /path/to/applications/mp-heat-transfer/mp-heat-transfer input_diffuse_IV_fine.json
```

### cut

```bash
mpirun -np 32 /path/to/applications/mp-heat-transfer/mp-heat-transfer input_cut_one_phase.json
mpirun -np 32 /path/to/applications/mp-heat-transfer/mp-heat-transfer input_cut_two_phase.json
```

## reference solution

```bash
mpirun -np 64 /path/to/applications/mp-heat-transfer/mp-heat-transfer input_cut_one_phase.json
```

### interface conforming mesh

The interface conforming mesh file `conforming_mesh.exodusii` was generated by cubit using the journal file `conforming_mesh_cubit_script.jou`.
