Detailed Description

template<int dim, typename number>
class MeltPoolDG::Functions::ChangedSignFunction< dim, number >

Wrapper function that flips the sign of another function.

This class takes a shared pointer to a dealii::Function and returns the negative of its value, gradient, and hessian. Time-related operations are forwarded to the underlying function.
