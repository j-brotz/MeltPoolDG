Detailed Description

template<int dim, typename number>
class MeltPoolDG::Heat::GusarovIntensityProfile< dim, number >

This class implements the laser heat source model of Gusarov et al. (2009). DOI: 10.1115/1.3109245

^ dim-1 2*R | <—> -—> x | | | | | | laser beam | | | power | | | | | |

------------------------------------—^ ++ ++ | ++ q ++ | layer thickness +++ +++ | ++++++ v

The laser beam is direction is assumed to be in the negative dim-1 direction.
