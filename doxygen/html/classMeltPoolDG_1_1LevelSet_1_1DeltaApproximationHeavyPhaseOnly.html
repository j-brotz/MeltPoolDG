Detailed Description

template<typename number>
class MeltPoolDG::LevelSet::DeltaApproximationHeavyPhaseOnly< number >

Dirac delta approximation function with a restriction towards the predominantely heavy material phase: 

       /
       | 2 * δ(φ)   if φ > 0.5

δ_w(φ) = | | 0 else. \

The function is dependent only on the heaviside representation of the level set φ (=indicator). The symmetric delta function δ(φ) is defined as the norm of the indicator gradient ∇φ:

δ = ||∇φ||
