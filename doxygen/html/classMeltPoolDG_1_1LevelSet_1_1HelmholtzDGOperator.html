Detailed Description

template<int dim, typename number>
class MeltPoolDG::LevelSet::HelmholtzDGOperator< dim, number >

This function applies the weak DG form of the Helmholtz operator n_ϕ - η_n*Δn_ϕ on a source vector. It is used for both the normal vector calculation and the calculation of the interface curvature.
