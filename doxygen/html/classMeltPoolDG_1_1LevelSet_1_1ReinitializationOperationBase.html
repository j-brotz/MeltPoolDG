Sets the boundary IDs where wetting boundary conditions should be applied.

This function forwards the provided list of boundary IDs to the reinitialization operation, indicating which boundaries are subject to wetting conditions.

Parameters
[in]wetting_bc_ids_inA list of boundary IDs where wetting conditions apply.
Remarks
This function is not implemented for the base class.
