Detailed Description

template<int dim, typename number>
class MeltPoolDG::ParticleIterator< dim, number >

STL-style bidirectional iterator over particles.

This iterator provides a uniform interface to traverse particles, whether they belong to a dealii::Particles::ParticleHandler or are stored in a PropertyPool. The interface is consistent for both cases and supports standard STL-style iterator operations, including: operator*, operator->, operator++, operator--, and comparison operators.

Dereferencing the iterator yields a ParticleAccessor object, which grants direct access to the particle's properties as defined in MeltPoolDG.
