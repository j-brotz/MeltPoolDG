Detailed Description

template<int dim, typename number>
class MeltPoolDG::Simulation::FixedMeltPool::FixedMeltPoolGeometry< dim, number >

This dealii::Frunction implements the fixed melt pool geometry as any of the LevelSetType representations. LevelSetType::signed_distance corresponds to eq. (37) in [1].
