Detailed Description

template<int dim>
class MeltPoolDG::Simulation::LaserMeltingSimonds::InitialConditionTemperature< dim >

Linear initial temperature profile in the vertical direction.

If top and bottom temperatures are identical, the function returns a constant initial temperature. Otherwise, it interpolates linearly between the bottom and top temperature values.
