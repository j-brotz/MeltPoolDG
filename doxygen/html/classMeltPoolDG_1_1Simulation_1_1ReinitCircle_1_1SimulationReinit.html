Detailed Description

template<int dim, typename number>
class MeltPoolDG::Simulation::ReinitCircle::SimulationReinit< dim, number >

A specific reinitialization simulation setup for a hyperrectangular domain with a hyperspherical zero level-set isosurface with an initially distorted level-set.
