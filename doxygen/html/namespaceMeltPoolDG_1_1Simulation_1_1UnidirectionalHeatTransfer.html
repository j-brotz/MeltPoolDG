Detailed Description

This example represents a simple test example for heat transfer. It offers the following configurations dependent on the input parameters:

case 1: emissivity > 0, convection_coefficient = 0 adiabatic +-----—+ | | adiabatic | | radiation | | +-----—+ adiabatic

case 2: emissivity = 0, convection_coefficient > 0 adiabatic +-----—+ | | adiabatic | | convection | | +-----—+ adiabatic

case 3: emissivity > 0, convection_coefficient > 0 adiabatic +-----—+ | | adiabatic | | radiation + convection | | +-----—+ adiabatic

case 4: emissivity = 0, convection_coefficient = 0 

       adiabatic
      +--------+
      |        |

adiabatic | T0=lin | adiabatic | | +-----—+ adiabatic
