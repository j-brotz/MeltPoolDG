Detailed Description

template<int dim, typename number>
template<typename T>
struct MeltPoolDG::AMR::AMRRegion< dim, number >::Model< T >

Concrete wrapper around a user-defined region type.

This class template models the Concept interface by forwarding calls to the underlying region object.

Template Parameters
TThe user-defined region type, which must implement bool point_inside(const dealii::Point<dim, number>&) const.
