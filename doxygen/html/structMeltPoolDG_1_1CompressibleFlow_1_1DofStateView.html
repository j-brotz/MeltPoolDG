Detailed Description

template<int dim, typename number, IsConservedStateCompatible< dim > Value>
struct MeltPoolDG::CompressibleFlow::DofStateView< dim, number, Value >

View providing access to the conserved variables stored in the underlying data structure. Besides direct access to the conserved variables, it enables computation of primitive variables, thermodynamic quantities, and material properties via the provided EOS and material data.

The underlying StateType must store the conserved variables in a tensor-like container indexed according to TensorStorageIndex<dim>.

Template Parameters
StateTypeType of the data structure storing the conserved variables.
