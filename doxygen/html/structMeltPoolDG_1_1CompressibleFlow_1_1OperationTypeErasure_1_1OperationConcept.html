Detailed Description

template<int dim, typename number>
struct MeltPoolDG::CompressibleFlow::OperationTypeErasure< dim, number >::OperationConcept

Abstract base class defining the interface for compressible flow operations.

This struct defines the pure virtual interface used for implementing type erasure in the CompressibleFlowOperation class. Concrete operation classes must implement this interface to be used with CompressibleFlowOperation.
