Detailed Description

template<int dim, typename number>
template<typename OperationType>
struct MeltPoolDG::CompressibleFlow::OperationTypeErasure< dim, number >::OperationModel< OperationType >

Concrete implementation of OperationConcept using a specific compressible flow operation type.

This struct wraps an actual compressible flow operation object (templated type) and forwards all interface calls defined in OperationConcept to it. This enables type-erased storage of operation implementations in CompressibleFlowOperation.

Template Parameters
OperationTypeThe concrete class type implementing the compressible flow operation logic.
